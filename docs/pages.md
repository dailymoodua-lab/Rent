# Страницы сайта

## Публичные страницы (marketing layout)

### `/` — Главная
- Hero-секция: заголовок, подзаголовок, CTA-кнопка "Выбрать авто"
- Краткая форма быстрого поиска (дата от/до, кнопка подбора)
- Блок "Почему мы": 4–6 преимуществ (страховка, круглосуточно, без залога и т.д.)
- Как это работает: 3 шага (выбрать → забронировать → получить)
- Популярные авто: 6–8 карточек из каталога
- Блок с отзывами (3–4 отзыва, слайдер)
- CTA-баннер: "Остались вопросы? Позвоните нам"

### `/catalog` — Каталог авто
- Панель фильтров: класс авто, КПП (авто/механика), тип кузова, ценовой диапазон
- Сортировка: по цене (возр./убыв.), по популярности
- Сетка карточек авто (фото, название, класс, КПП, цена/сутки, кнопка "Забронировать")
- Пагинация или infinite scroll
- Счётчик найденных авто

### `/car/[slug]` — Карточка авто
- Галерея фото
- Название, класс, характеристики (КПП, привод, объём, мест, год)
- Цена за сутки / за 3 дня / за неделю / за месяц
- Описание авто
- Что включено в аренду (страховка, помощь на дороге и т.д.)
- Форма бронирования (дата от/до, расчёт суммы)
- Похожие авто (3–4 карточки)

### `/booking` — Оформление бронирования
- Шаг 1: Выбранное авто + даты (подтверждение)
- Шаг 2: Данные клиента (имя, телефон, email, серия паспорта)
- Шаг 3: Итого + оплата (онлайн или при получении)
- Индикатор шагов

### `/booking/success` — Подтверждение брони
- Номер брони
- Детали: авто, даты, адрес получения
- Кнопки: "Перейти в личный кабинет", "На главную"

### `/account` — Личный кабинет (redirect → /account/bookings)
Требует авторизации.

### `/account/bookings` — Мои брони
- Список активных и прошлых броней
- Статус: ожидает подтверждения / активна / завершена / отменена
- Кнопка "Отменить" для активных

### `/account/profile` — Мой профиль
- ФИО, телефон, email, дата рождения
- Редактирование данных
- Смена пароля

### `/about` — О нас
- История компании / кто мы
- Команда (фото + имя + должность)
- Наш автопарк в цифрах (кол-во авто, лет на рынке, клиентов)
- Лицензии и сертификаты

### `/conditions` — Условия аренды
- Требования к водителю (возраст, стаж, документы)
- Залог
- Правила использования авто
- Страховка: что покрывает, что нет
- Штрафы и ответственность
- FAQ-аккордеон

### `/services` — Другие услуги
- Аренда с водителем
- Трансфер в аэропорт
- Прокат для свадьбы / мероприятий
- Доставка авто

### `/business` — Для бизнеса
- Корпоративная аренда: условия, скидки
- Долгосрочная аренда (от 1 месяца)
- Форма заявки для юридических лиц

### `/buyout` — Под выкуп
- Описание схемы выкупа
- Как это работает: 4–5 шагов
- Калькулятор (сумма, срок → ежемесячный платёж)
- Форма заявки

### `/reviews` — Отзывы
- Список отзывов клиентов (фото, имя, дата, текст, оценка)
- Фильтр по оценке
- Форма добавления отзыва (для авторизованных)

### `/blog` — Блог
- Сетка статей (обложка, заголовок, дата, краткое описание)
- Категории/теги
- Пагинация

### `/blog/[slug]` — Статья блога
- Заголовок, дата, автор
- Тело статьи (MDX или rich text)
- Блок "Похожие статьи"

### `/contacts` — Контакты
- Адрес офиса + время работы
- Телефоны, email
- Карта (Google Maps embed)
- Форма обратной связи

## Системные страницы

### `/login` — Вход
- Форма: email + пароль
- Кнопка "Войти через Google"
- Ссылка "Зарегистрироваться"

### `/register` — Регистрация
- Форма: имя, email, телефон, пароль
- Согласие с условиями

### `/404` — Страница не найдена
- Сообщение + кнопка "На главную"
